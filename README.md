Interactive Gestalt Laws Viewer with Progress Tracking

📖 Description

This project is a simple interactive web application developed as part of an HCI (Human-Computer Interaction) assignment. It demonstrates the concept of Direct Manipulation, specifically:

Incremental user actions
Immediate visual feedback
Continuous system response

The application allows users to explore definitions of six Gestalt laws while tracking their progress visually.
🎯 Objectives
Implement Direct Manipulation principles in UI design
Provide real-time feedback for user actions
Enhance user engagement through interactivity
Track user progress dynamically

🧠 Features
✅ Clickable menu of six Gestalt laws
✅ Highlighting of selected law (visual feedback)
✅ Display of definitions instantly
✅ Dynamic progress bar
✅ Percentage indicator of viewed laws
✅ Smooth and responsive UI

🖥️ Technologies Used
HTML
CSS
JavaScript

🧩 Gestalt Laws Included
Law of Proximity
Law of Similarity
Law of Continuity
Law of Closure
Law of Figure/Ground
Law of Common Fate

⚙️ How It Works
User clicks on a Gestalt law
The selected item is highlighted
Definition is displayed instantly
Progress bar updates
Percentage of completed views increases

This demonstrates incremental action with immediate feedback, a key principle of Direct Manipulation.

📊 Example Functionality
If 3 out of 6 laws are viewed → Progress = 50%
Progress bar updates in real time

📚 Learning Outcomes
Practical implementation of Direct Manipulation
UI/UX design principles
Event handling in JavaScript
Dynamic DOM updates
